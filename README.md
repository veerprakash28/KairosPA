# ChronosPA - Personal Assistant & Reminder Dashboard

A premium, glassmorphic desktop personal assistant dashboard web application. It handles task scheduling, notifications (native desktop alerts + chimes), carrying forward pending tasks, and parses conversational natural language commands to add reminders.

## Features

- **System Alerts**: Native browser desktop notifications with synthesized chime sound.
- **Natural Language Input**: Type instructions directly to the PA sidebar:
  - *"remind me to stand up in 10 minutes"*
  - *"add water break at 15:00"*
  - *"postpone meeting to tomorrow"*
- **Carry Forward**: Shift overdue tasks to tomorrow with a single click.
- **Glassmorphic UI**: High-fidelity dark mode dashboard with glowing status rings, visual timelines, and weekly/monthly calendar.
- **Zero Dependencies**: Pure HTML, CSS, and JS. Runs locally without build overhead.

## Setup & Running

1. Open `index.html` directly in any web browser (Double-click `index.html` or run a local static server).
2. When prompted, click **Allow** to enable desktop notifications.
3. Make sure sound is unmuted in the tab to hear the synthesized notification chime.
